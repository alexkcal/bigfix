Title: Adobe Acrobat Pro DC 15.009.20077 - Adobe Acrobat Pro DC 15.x - Mac OS X
Additional Notes: 15.009.20077 Out Of Cycle Update (Continuous Track), October 29, 2015

The version of Acrobat DC after installing this update is 15.009.20077. BigFix detects it as "15.009.20077" which is the value used in the Relevance

----------
DESCRIPTION
----------
Use this action to update Adobe Acrobat Pro DC to version 15.009.20077. This action is relevant on Macs running Mavericks or newer with a previous version of Adobe Acrobat Pro DC installed.

Adobe Acrobat Pro DC Update 15.009.20077 Out Of Cycle Update (Continuous Track), October 29, 2015

Action 1: Use this when Acrobat Pro DC is not running otherwise the update will fail. Acrobat Pro DC needs to be closed in order for the update to succesfully install. If you would like to forcibly close Acrobat Pro DC, then use Action 2.

Action 2: Use this to forcibly close Acrobat Pro DC by killing the process, and then proceed with installing the update.

----------
RELEVANCE
----------
mac of operating system

system version >= "10.9.0"

exists application whose (name of it = "Adobe Acrobat.app" and version of it < "15.9.20077" AND version of it >= "15.0")

----------
ACTION SCRIPT
----------
ACTION 1 - Click here to initiate the deployment process. 
prefetch AcrobatDCUpd1500920077.pkg sha1:0bd3e9c6494757440b167f830e43f795013c32cd size:168481038 http://ardownload.adobe.com/pub/adobe/acrobat/mac/AcrobatDC/1500920077/AcrobatDCUpd1500920077.pkg sha256:effc530b5b15e5e3b48825d5cd59afd042017fd6c76151a5b4668686de2fdf56

// Delete the Acrobat DC Update PKG in case it already exists
wait /bin/sh -c "rm '/private/tmp/AcrobatDCUpd1500920077.pkg' "

// Kill all Acrobat processes to forcibly close Acrobat DC
// Uncomment the next line to forcibly close Acrobat DC by killing the process
// wait /bin/sh -c "/usr/bin/killall 'AdobeAcrobat' "

// Move the downloaded Acrobat DC Update PKG to the temp directory
move "{posix path of file "AcrobatDCUpd1500920077.pkg" of folder "__Download" of client folder of current site}" "/private/tmp/AcrobatDCUpd1500920077.pkg"

// Install the Acrobat DC update
wait installer -pkg "/private/tmp/AcrobatDCUpd1500920077.pkg" -target /

// Delete the Acrobat DC Update PKG in the temp directory
wait /bin/sh -c "rm '/private/tmp/AcrobatDCUpd1500920077.pkg' "

// Uncomment the next line to require a restart after installing the update
// action requires restart
